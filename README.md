PROJECT DESCRIPTION:
-
A simple browser-based word guessing game inspired by Wordle.
Built as part of a GitHub Pages website using separate HTML, CSS, and JavaScript files.
Lets the player guess a hidden 5-letter word in up to 6 tries.
Uses keyboard input, game states, feedback colors, and restart functionality.
Includes a basic home page, game page, responsive styling, and a few extra features like stats, timer, and simple animations.
